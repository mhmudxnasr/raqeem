# Raqeem v0.0.1 (Initial Release)

Welcome to the first official release of **Raqeem**, your personal finance companion.

## 🚀 Key Features
- **Modern Dashboard**: Unified view of your financial state.
- **AI-Driven Insights**: Get intelligent feedback on your spending.
- **Secure Synchronization**: Real-time sync with Supabase backend.
- **Biometric Security**: Protect your sensitive information with Android Biometrics.

## 📥 Installation Instructions

This is a direct APK release. To install:

1. **Download**: Click on the `app-release.apk` asset below.
2. **Authorize**: If you see a security warning, go to **Settings** -> **Allow from this source** (or "Install Unknown Apps").
3. **Install**: Return to the APK and complete the installation.
